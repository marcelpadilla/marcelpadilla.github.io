Nailing the Bunny

A private side project, 2020
Marcel Padilla
TU Berlin

Abstract

Grab some white board, hammer some nails into it, and connect the nails to form the visualisation of a 3D mesh.

I present here the short story of how this project turned out.

Connecting the Nails

Once all the nails were hammered in, I could connect all each of them using lines. Originally, I was set to use a thing black string for this. I quickly noticed that that would take way too much time. So instead, I drew the lines with a black fine liner.

Connecting all the nails using black lines.
