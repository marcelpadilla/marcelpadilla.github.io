Hyperbolic Tiling {3,7}

A private side project by Marcel Padilla - TU Berlin - 2020

Intro

Here are some documentations on my attempt to generate a hyperbolic tiling. The goal was to set up a system where I can use any texture/video to be displayed inside the tiling. With a bit of creativity, we could then make images in the style of M.C. Escher‚Äôs circle limit.

Mission Statement

Given any texture of a triangle in the Klein disc model, map it into each triangle of the {3,7} tiling of the Poincar√© disk model.

Solution

  • Create the basic reference mesh for the tiling
  • Create a refined textured mesh of a single tile
  • Expand the refined tile using the reference mesh
  • Place a circle of constant color behind the tiling for fine details at the edge
  • Be aware of floating point issues near the edge
